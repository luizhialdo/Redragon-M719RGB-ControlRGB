# Redragon-M719RGB-ControlRGB
I don't have much technical knowledge, but I reverse-engineered and built this simple program to control the RGB lighting on the Redragon M719rgb-pro mouse.
Choose a version (CLI or GUI).
To compile, you need to run the build.sh script with sudo. To run the compiled program without using sudo, you need to run the "setup_udev.sh" script.
